Detailed Description

EDGE = Borda do elemento

Arestas das faces, compostas por dois vertices;

Parameters
tipo=-1 inflow; = 0 no-flow; =+1 outflow; = 2 interior;
Na= number of the initial mode
Nb= number of the final mode
elemento[2]= numbers of the elements sharing the border
num_local[2]= side numbers the border assumes in the elements
sinal[2]= sign the border has in the elements; -1 if the border is followed backwards;
normal[2]=components of the normal vector;
comprimento= length of the border;
*pdir= pointer to the array containing the Dirichlet values of pressure
*sdir= pointer to the array containing the Dirichlet values of saturation
gbtrbind= global trace border index for the traces on the sides
part_num= partition number
flux_w= flux of w through the border
flux_n= flux of non-wetting fluid through the border
